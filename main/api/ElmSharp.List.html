Class List

    Definition

    Namespace:
    ElmSharp
    Assembly:
    ElmSharp.dll

    It inherits Layout. The List is a widget that aims to display a simple list item which has 2 icons, 1 text, and can be selected. For more robust lists, GenList should probably be used.
