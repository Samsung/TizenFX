Class TextureSet

    Definition

    Namespace:
    Tizen.NUI
    Assembly:
    Tizen.NUI.dll

    TextureSet is a handle to an object that specifies the set of images used as textures by a renderer.
    The images have to be ordered in the same order they are declared in the shader.
