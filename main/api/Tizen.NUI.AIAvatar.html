Namespace Tizen.NUI.AIAvatar

    Classes

    AnimationLoader

    The AnimationLoader class provides methods to load and manage body and face motion data from specified resources.

    AnimatorBase

    The base class for all animator implementations. Provides basic functionality for adding, playing, stopping, pausing, and removing animations.

    AnimatorChangedEventArgs

    This class provides arguments for handling avatar motion change events. The previous state of the avatar's motion. The current state of the avatar's motion.

    EmotionAnimator

    The EmotionAnimator class extends SerialAnimator and provides functionality to play animations based on emotions. It manages emotion configurations, generates expression data animations, and loads necessary resources.

    LipSyncer

    LipSyncer class manages the synchronization of an avatar's mouth movements with audio input. It handles the creation, enqueuing, playing, pausing, and stopping of animations based on lip-sync data.

    MotionInfo

    The AnimationInfo class manages animation data for an Avatar, including motion data and names. It is not meant to be directly edited by users or editors.

    ParallelAnimator

    The ParallelAnimator class extends the AnimatorBase class and provides methods to play, stop, and pause animations in parallel.

    SerialAnimator

    Represents an animator that plays animations in a serial manner.
