Class Log

    Definition

    Namespace:
    Tizen
    Assembly:
    Tizen.Log.dll

    Provides methods to print log messages to the Tizen logging system. Depending on products, some priorities (e.g., Vervose and Debug) can be disabled by default to prevent too many logs.
