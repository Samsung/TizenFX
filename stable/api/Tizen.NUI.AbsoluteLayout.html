Class AbsoluteLayout

    Definition

    Namespace:
    Tizen.NUI
    Assembly:
    Tizen.NUI.dll

    This class implements a absolute layout, allowing explicit positioning of children. Positions are from the top left of the layout and can be set using the View.Position and alike.
