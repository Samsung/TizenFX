Class LipSyncer

    Definition

    Namespace:
    Tizen.NUI.AIAvatar

    LipSyncer class manages the synchronization of an avatar's mouth movements with audio input. It handles the creation, enqueuing, playing, pausing, and stopping of animations based on lip-sync data.
